📅 EventManager

A community event management platform built with ASP.NET Core

ℹ️ GitHub Pages Note:
This is a server-side ASP.NET application. GitHub Pages only hosts static sites. For the full experience, run locally or deploy to a server.

To run locally:

  1. Clone this repository
  2. Run dotnet run
  3. Open http://127.0.0.1:5002
  4. Login: admin / Admin@123 or user / User@123
🚀 To deploy: Use Azure App Service, Render, Heroku, or any .NET hosting provider.
